Swift - 字典中的增删改,遍历与合并

2016/11/12

增删改

var dict:[String:Any] = ["name" : "小明", "age" : 18, "sex":"man"]

// 新增
dict["title"] = "老大"
print(dict)

// 修改
dict["name"] = "赵四"
print(dict)

// 删除
dict.removeValue(forKey: "age")
print(dict)

遍历,合并

// 遍历
for e in dict {
    print(e)
}

for (k,v) in dict {
    print("key:\(k)  value:\(v)")
}
// 合并
var dict1:[String:Any] = ["name" : "李明", "age" : 18, "sex":"m"]
var dict2:[String:Any] = ["name" : "韩梅梅", "age" : 16, "sex":"w"]
// 如果把 dict2 合并到 dict1
for e in dict2 {
    dict1[e.key] = e.value   //dict2[e.key]
}
print("----------\n\(dict1)")

Swift - 数组中的增删改与合并

2016/11/12

第一篇写了数组遍历的几种情况,这篇主要就是数组中的增删改,也是开发中常用的

OC中数组分为可变(NSMutableArray)和不可变(NSArray) 相应的在Swift中就是 var & let

##追加元素
首先定义一个可变数组

var array = ["张三","李四","王五"]

追加一个元素

// 追加元素
array.append("赵六")
print(array)

##修改元素
通过下标进行修改

array[0] = "赵四"
print(array)

结果:

["赵四", "李四", "王五", "赵六"]

##删除

// 删除元素
array.remove(at: 0)
print(array)

结果:

["赵四", "李四", "王五", "赵六"]
["李四", "王五", "赵六"]

remove 的方法由很多,移除首/尾,移除范围...

array.removeAll(keepingCapacity: true)
print("数组中的元素\(array) 数组容量\(array.capacity)")

结果:

数组中的元素[] 数组容量6

##数组合并

var arr0 = [1,2,3]
let arr1 = [3,4,5]
arr0 += arr1
print(arr0)

结果:

[1, 2, 3, 3, 4, 5]

注意: 要合并的数组的类型必须一致,如果不一致会出错. 比如:

var arr0:[Any] = [1,2,3]
let arr1:[Any] = ["2",3,4,5]
arr0 += arr1
print(arr0)

Swift - 数组遍历

2016/11/12

最近没怎么更新,主要是最近刚入职新公司.正在熟悉环境中... 好了题目是Swift - 数组遍历 主要是项目要使用 Swift 开发,所以需要熟悉下语法.

##传统的下标遍历方法

let array = ["张三","李四","王五"]

// 第一种 下标遍历
for i in 0..<array.count {
    print(array[i])
}

##for in 遍历元素

// 第二种 for in 遍历元素
for s in array {
    print(s)
}

##enum block遍历

// 第三种 enum block 遍历
for e in array.enumerated() {
    print(e)
}

结果:

(0, "张三")
(1, "李四")
(2, "王五")

我们可以看下e的类型--元组

##元组方法遍历下标和内容
由上面第三种方法我们可以知道 e 的类型是元组,那么同样我们可以直接使用元组

// 第四种 元组使用
for (n,s) in array.enumerated() {
    print("\(n) \(s)")
}

结果:

0 张三
1 李四
2 王五

##反序遍历

// 第五种 反序遍历
for i in array.reversed() {
    print(i)
}

结果

王五
李四
张三

##反序遍历 带索引

// 第六种 反序遍历 带索引
for (n,s) in array.enumerated().reversed() {
    print("\(n) \(s)")
}

结果:

2 王五
1 李四
0 张三

注意: 先枚举再反序

总结:Swift 太灵活了...
