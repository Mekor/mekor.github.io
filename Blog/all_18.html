Swift - 面向协议编程

2016/11/24

跟全部写在base中比较:

  1. 代码没有增多,却更复用
  2. 面向协议(接口),而不是实现,充分解耦
  3. 静态类型检查版主在编译时发现问题
  4. 写代码像搭积木,先设计接口,再逐一实现
  5. 依赖少,更容易调试
protocol Hello {
    func sayHello()
}

extension Hello {
    func sayHello() {
        print("Hello!")
    }
}

struct Person: Hello {
    
}

let p = Person()
p.sayHello()

创建一个协议,并且做一个默认实现.使用的时候可以直接调用

Swift - 函数式编程

2016/11/23

一般情况:

struct User {
    func login(password: String){
        print(password)
    }
}

let passwd = "123123123"
let usr = User()

usr.login(password: passwd)

函数式编程

struct User {
    func login(password: String){
        print(password)
    }
}

let passwd = "123123123"
let usr = User()

User.login(usr)(password: passwd)

由此可见: usr.login(password: passwd) == User.login(usr)(password: passwd)

如果忽略外部参数

struct User {
    func login(_ password: String){
        print(password)
    }
}

let passwd = "123123123"
let usr = User()
usr.login(passwd)
User.login(usr)(passwd)

这样就更简洁了

##柯里化(Currying)

// 柯里化(Currying)

func addOne(num:Int) ->Int {
    return num + 1
}

print(addOne(num: 10))

func addTo(_ adder:Int) ->(_ num:Int) -> Int {
    return {
        num in
        return num + adder
    }
}

let addTwo = addTo(2)

print(addTwo(6))

// 更简单的调用方式
print(addTo(9)(1))

Swift - 使用代理和闭包传值

2016/11/20

##代理传值

声明协议:

protocol FunctionButtonViewDelegate {
    func gotoViewController(vcString:String)
}

var delegate :FunctionButtonViewDelegate?

这里?就具有weak的作用

传值:

delegate?.gotoViewController(vcString: button.currentTitle!)

实现代理方法

extension PersonalCenterViewController:FunctionButtonViewDelegate {
    //MARK: - 跳转到对应的功能模块
    func gotoViewController(vcString: String) {
        print(vcString)
    }
}

##闭包传值

声明:

var callback : ((Int?) -> Void)?

传值

if let cb = callback {
    cb(200)
}

实现:

bottomView.callback = { (num)->Void in
    print(num ?? 0)
}

Swift - 在swift中如何使用 hexcolor 值表示颜色

2016/11/20

结果来自How to use hex colour values in Swift, iOS

extension UIColor {
    convenience init(hexString: String) {
        let hex = hexString.trimmingCharacters(in: CharacterSet.alphanumerics.inverted)
        var int = UInt32()
        Scanner(string: hex).scanHexInt32(&int)
        let a, r, g, b: UInt32
        switch hex.characters.count {
        case 3: // RGB (12-bit)
            (a, r, g, b) = (255, (int >> 8) * 17, (int >> 4 & 0xF) * 17, (int & 0xF) * 17)
        case 6: // RGB (24-bit)
            (a, r, g, b) = (255, int >> 16, int >> 8 & 0xFF, int & 0xFF)
        case 8: // ARGB (32-bit)
            (a, r, g, b) = (int >> 24, int >> 16 & 0xFF, int >> 8 & 0xFF, int & 0xFF)
        default:
            (a, r, g, b) = (255, 0, 0, 0)
        }
        self.init(red: CGFloat(r) / 255, green: CGFloat(g) / 255, blue: CGFloat(b) / 255, alpha: CGFloat(a) / 255)
    }
}

使用方法:

let darkGrey = UIColor(hexString: "#757575")
